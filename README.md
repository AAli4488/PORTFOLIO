# Ayesha Siddiqa Ali — Portfolio

A personal portfolio website built from scratch with HTML and CSS. Showcases my work in web development and AI/ML.

🔗 **Live site:** [ayesha-portfolio.netlify.app]ayesha-siddiqa-portfolio.netlify.app

---

## About

I'm a final-year Computer Science student in India, building at the intersection of AI/ML and modern web development. This portfolio is my first deployed project — written by hand to learn the fundamentals before moving on to frameworks.

---

## Features

- Fully responsive layout (desktop, tablet, mobile)
- Dark theme with custom color palette and typography
- Sticky navigation with smooth scroll
- Hero section with call-to-action buttons
- Skills, Projects, and Contact sections
- Hover effects and micro-interactions
- Mobile-first media queries

---

## Tech Stack

- **HTML5** — semantic structure
- **CSS3** — Flexbox, Grid, custom properties, media queries
- **Google Fonts** — Space Grotesk + JetBrains Mono

No frameworks. No libraries. Just the fundamentals.
